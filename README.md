# Pomodoro-Timer
# Project URL https://pomodoro-timer-two-bice.vercel.app/

A clean and simple web-based Pomodoro Timer to help you stay focused and productive. This application allows you to switch between focused work sessions and short or long breaks, following the principles of the Pomodoro Technique.

The interface is designed to be intuitive, minimalist, and distraction-free, with a responsive layout that works on both desktop and mobile devices.

## Features

- **Three Timer Modes:** Easily switch between a 25-minute Pomodoro session, a 5-minute short break, and a 10-minute long break.
- **Simple Controls:** Straightforward START and STOP buttons to manage the timer.
- **Audio Notifications:** An audible alarm plays when a session or break is complete.
- **Clean UI:** A modern, dark-themed interface with a circular timer display.
- **Responsive Design:** The layout automatically adjusts for optimal viewing on any screen size.
- **No Dependencies:** Built with pure HTML, CSS, and JavaScript.

## How to Use

1. Open the `index.html` file in your favorite web browser.
2. Select your desired timer mode: **Pomodoro**, **Short Break**, or **Long Break**. The Pomodoro timer is selected by default.
3. Click the **START** button to begin the countdown.
4. Click the **STOP** button to pause the timer at any time.
5. When the timer reaches `00:00`, an alarm will sound to notify you.
